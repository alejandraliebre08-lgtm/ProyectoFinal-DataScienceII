# Predicción Salarial en IA y Ciencia de Datos

Proyecto Final - Data Science II | Coderhouse 2026

## Descripción
Análisis del mercado laboral en IA/DS para predecir salarios mediante Machine Learning.
Dataset de 10.345 registros con variables de experiencia, país, rol y habilidades técnicas.

## Modelos entrenados
- Random Forest (R² 0.9392)
- Regresión Lineal (R² 0.9444) ← Mejor modelo
- Random Forest Optimizado (R² 0.9278)

## Resultado
Regresión Lineal con R² 0.9444 y MAE $6,008 USD (error del 5.3%)
